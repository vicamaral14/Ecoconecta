# 🌱 EcoConecta

Sistema Web desenvolvido para conectar cidadãos, empresas, ONGs e cooperativas por meio da logística reversa, incentivando o descarte correto de materiais recicláveis e promovendo impacto social e ambiental.

---

## 📖 Sobre o Projeto

O **EcoConecta** é uma plataforma que facilita o cadastro de doações de materiais recicláveis e conecta doadores a coletores, permitindo o acompanhamento de todo o processo de coleta de forma simples, organizada e transparente.

O objetivo do sistema é incentivar práticas sustentáveis, fortalecer a economia circular e contribuir para a preservação do meio ambiente.

---

## 🎯 Objetivos

- Incentivar o descarte correto de resíduos recicláveis;
- Facilitar a comunicação entre doadores e coletores;
- Promover a logística reversa;
- Acompanhar todo o ciclo da doação;
- Gerar impacto social e ambiental positivo.

---

## 🚀 Funcionalidades

- Cadastro de usuários
- Login e autenticação
- Cadastro de materiais recicláveis
- Registro de doações
- Consulta de materiais disponíveis
- Reserva de materiais por coletores
- Atualização do status da coleta
- Histórico de doações
- Gerenciamento de usuários

- MySQL

- XAMPP (recomendado)
- Navegador Web

---

## ▶️ Como Executar

### 1. Clone o repositório

```bash
git clone https://github.com/SEU-USUARIO/EcoConecta.git
```

### 2. Entre na pasta

```bash
cd EcoConecta
```

### 3. Copie o projeto para a pasta do XAMPP

```
C:\xampp\htdocs\
```

### 4. Inicie o Apache e o MySQL

Abra o painel do XAMPP e inicie os serviços:

- Apache
- MySQL

### 5. Crie o banco de dados

Abra o phpMyAdmin.

Crie um banco chamado:

```sql
CREATE DATABASE ecoconecta;
```

Depois importe o arquivo SQL localizado na pasta do projeto.

### 6. Execute

Abra o navegador:

```
http://localhost/EcoConecta
```

---

## 🗄 Banco de Dados

O sistema utiliza MySQL para armazenar informações sobre:

- Usuários
- Doações
- Materiais
- Coletas
- Reservas
- Histórico

---

## 🔄 Fluxo do Sistema

1. O usuário realiza o cadastro.
2. Efetua login.
3. Cadastra um material reciclável.
4. O material fica disponível para coleta.
5. Um coletor realiza a reserva.
6. A coleta é confirmada.
7. O sistema registra todo o histórico.
